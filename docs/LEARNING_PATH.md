# HealthSynth Learning Path

HealthSynth is designed as a sandbox for learning commercial life sciences analytics.

## Recommended Path

1. `01_running_healthsynth.ipynb`  
   Learn how to generate data using Python, CLI-style options, CSV, DuckDB, and YAML profiles.

2. `02_market_profiles.ipynb`  
   Compare different commercial environments generated from YAML profiles.

3. `03_competitor_products.ipynb`  
   Understand product metadata, manufacturers, brand types, and baseline market share.

4. `04_duckdb_analytics.ipynb`  
   Practice SQL analytics on generated commercial datasets.
